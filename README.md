### 介绍

- 本主题基于 zhang-light 主题修改而来，属性通用
- 绿色护眼，顶栏深色绿，侧栏渐变色，文档树毛玻璃效果，显得有活力

### 属性介绍

- 详见“zhang-light”主题集市说明

### 专属自定义属性

- 绿象专属块自定义属性
  - f=blt 是绿色配色的背景样式
  - table=no 如果觉得表格太话，可以用这个属性，仅保留边框，去掉底色
