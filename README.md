## 介绍

- 本主题经群友反馈颜色偏绿，所以进行了重构：加深了正文颜色，更加护眼，侧栏设置为了灰色，标签页模仿 Chrome 的**梯形标签页**
- 本主题基于 Chrome-A4 主题修改而来，自定义属性和 Zhang-Light 主题一致
- 后续会持续进行维护，请放心使用
- 暂时没有加入 JS 脚本，下次更新加入
